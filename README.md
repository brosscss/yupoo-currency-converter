# Yupoo Currencies Converter

## Overview

Yupoo Currencies Converter is a Chrome extension that provides real-time currency conversion. It automatically fetches the latest exchange rates and converts prices from Yuan to a user-selected currency on web pages. This extension is ideal for users who frequently deal with international e-commerce sites.

## Features

- Real-time currency conversion.
- Supports multiple currencies with corresponding symbols.
- User-friendly settings to save preferred currency.

## Installation

### From Chrome Web Store

1. Visit the Chrome Web Store at [Yupoo Currencies Converter](https://chromewebstore.google.com/detail/yupoo-currencies-converte/dfdfhiljhhamcikbllphacpchjffaimk?hl=fr).
2. Click "Add to Chrome" to install the extension.

### Importing to Chrome

1.Open Chrome and navigate to chrome://extensions/.
2.Enable "Developer mode" at the top right.
3.Click "Load unpacked" and select the dist directory from the build process.
4.The extension should now be installed and visible in your Chrome extensions list.

### Usage

After installation, the extension will automatically convert prices from Yuan to your selected currency on supported web pages.
